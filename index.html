Responsive Web Design Principles

This page was created to keep track of new web design principles!

Media Query

A media query changes the elements based on the size of the viewport.

It allows the developer more control over the size of elements on a page whether on a small mobile screen or a large family TV.

Responsive Images

With just a couple simple css attributes, an image can automatically adjust to best fit the viewport.

NOTE: This will be the only responsive image on the page to prove a point.

Retina Displays

In newer displays pixels are so densely packed that the human eye can't make out any individual pixel at standard viewing distance.

Apple has coined this term "retina display" for marketing purposes back in 2012.

"Due to the difference in pixel density between a "Retina" and "Non-Retina" displays, some images that have not been made with a High-Resolution Display in mind could look "pixelated" when rendered on a High-Resolution display." -freeCodeCamp

As a result, it's good practice to scale images down that appear too grainy or pixelated.

Responsive Typography

vh & vw

Just like images, text should respond correctly to a variety of screen sizes. This can be accomplished with the viewport height <vh> and viewport width <vw> elements

vmax & vmin

Sometimes developers need to adjust elements to respond to maximum or minimum bounds. The <vmax> and <vmin> units can help achieve this by taking both width and height into consideration.
